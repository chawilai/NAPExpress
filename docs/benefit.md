# AutoNAP — สรุปความคุ้มค่าและประโยชน์ของระบบบันทึก NAP Plus อัตโนมัติ

## ภาพรวม

AutoNAP คือระบบบันทึกข้อมูลลงเว็บ NAP Plus ของ สปสช. แบบอัตโนมัติ แทนที่เจ้าหน้าที่จะต้องนั่งกรอกฟอร์มทีละคน ระบบจะรับข้อมูลจาก CAREMAT แล้วกรอกให้อัตโนมัติผ่าน browser — รองรับ 3 ฟอร์ม: Reach RR, VCT, และ Request Lab HIV

---

## 2. ความถูกต้อง (Accuracy)

### ปัญหาที่พบจากการกรอกด้วยมือ

| ปัญหา | คนกรอก | AutoNAP |
|-------|--------|---------|
| กรอก PID ผิด / สลับตัวเลข | เกิดบ่อย | ไม่มี — ดึงจากฐานข้อมูลโดยตรง |
| เลือก KP ผิดกลุ่ม | เกิดได้ | ไม่มี — map อัตโนมัติจากข้อมูลต้นทาง |
| วันที่ให้บริการผิด | เกิดได้ (copy/paste ผิดบรรทัด) | ไม่มี — ใช้ service_date จาก CAREMAT ตรง |
| ลืมเลือก checkbox (ความรู้/PPE/ปัจจัยเสี่ยง) | เกิดบ่อย (เพราะ field เยอะ) | ไม่มี — เลือกครบทุก field ตาม business rule |
| กรอกจำนวนถุงยางผิด | เกิดได้ | ไม่มี — default ตามที่กำหนด |
| ลืมกด save / กด back โดนข้อมูลหาย | เกิดได้ | ไม่มี — script ทำงานต่อเนื่อง |

### อัตราความถูกต้อง (ประมาณการ)

| | คนกรอก | AutoNAP |
|---|---|---|
| ความถูกต้องของ PID | ~95-98% | **100%** |
| ความถูกต้องของ KP matching | ~90-95% | **100%** |
| ความครบถ้วนของ fields | ~85-95% | **100%** |
| **โดยรวม** | **~85-95%** | **~99-100%** |

## 3. ความทันเวลา (Timeliness)

### ปัญหาของการกรอกด้วยมือ

- เจ้าหน้าที่มีงานอื่นต้องทำ → **ข้อมูลค้างสะสม** บันทึกไม่ทัน
- บันทึกย้อนหลังเป็นสัปดาห์/เดือน → **ข้อมูลไม่ทันสมัย** สำหรับ monitoring
- เร่งกรอกตอนใกล้ deadline → **เพิ่มโอกาสผิดพลาด**
- ข้อมูลที่ไม่ทันเวลา → **กระทบการเบิกจ่ายงบประมาณ**

### AutoNAP แก้ปัญหานี้

| | คนกรอก | AutoNAP |
|---|---|---|
| เวลาตั้งแต่ให้บริการ → บันทึก NAP | วัน-สัปดาห์ | **ภายในวันเดียวกัน** |
| สามารถบันทึกทันที่หลังจบบริการ | ไม่ได้ (ต้องรอคิว) | ได้ — กดส่งเมื่อไรก็ได้ |
| ข้อมูลพร้อมสำหรับ reporting | ล่าช้า | **near real-time** |
| ข้อมูลพร้อมสำหรับเบิกจ่าย | อาจไม่ทัน | **ทันเวลาเสมอ** |

## 4. ความครบถ้วน (Completeness)

### ปัญหาที่พบ

- คนกรอก **ข้าม records** ที่กรอกยาก (เช่น ชื่อซ้ำ ต้อง search หลายรอบ)
- **ลืมบันทึก** บาง records เพราะข้อมูลเยอะ
- **ลืม Request Lab** หลังจากบันทึก VCT เสร็จ
- ข้อมูลที่ขาด → **เบิกจ่ายไม่ได้** / reporting ไม่ครบ

### AutoNAP แก้ปัญหานี้

| | คนกรอก | AutoNAP |
|---|---|---|
| อัตราการบันทึกครบ | ~90-95% | **100%** (ทุก record ที่ส่งมา) |
| VCT + Lab ต่อเนื่อง | อาจลืม Lab | **อัตโนมัติ** — ทำ Lab ต่อทันที |
| Records ที่หลุด | ต้องตาม manually | รายงาน error กลับ → ส่งซ้ำได้ |
| Callback update CAREMAT | ต้องกลับมา update เอง | **อัตโนมัติ** — ได้ NAP code กลับทันที |

## 5. ลดภาระงานเจ้าหน้าที่

### ก่อนใช้ AutoNAP

1. เจ้าหน้าที่ต้อง login NAP Plus
2. กรอกฟอร์มทีละคน ซ้ำๆ เหมือนกัน
3. Copy PID จาก Excel → paste → กด search → กรอก → save → ทำซ้ำ
4. ใช้เวลาวันละ 4-8 ชั่วโมง เฉพาะงาน data entry

### หลังใช้ AutoNAP

1. เลือก records ที่ต้องการบันทึก
2. กดปุ่ม "ส่งบันทึก AutoNAP"
3. สแกน ThaiD QR 1 ครั้ง
4. รอ — ระบบทำงานเอง
5. ได้ผลลัพธ์ + รหัส NAP กลับมาอัตโนมัติ

**เจ้าหน้าที่ใช้เวลาจริงแค่ ~2 นาที** (เลือก records + สแกน QR) — ที่เหลือ AutoNAP ทำให้ทั้งหมด

## 7. ความปลอดภัยและการตรวจสอบ

| คุณสมบัติ | รายละเอียด |
|----------|-----------|
| Login ปลอดภัย | ใช้ ThaiD QR scan (ไม่เก็บ password) |
| Audit log | ทุก request ถูกบันทึกเป็น JSON file ตรวจสอบย้อนหลังได้ |
| Real-time monitoring | ดู progress ผ่าน WebSocket (Ably) แบบ real-time |
| Error reporting | error message จาก NAP Plus ส่งกลับ CAREMAT ทันที |
| ไม่แก้ไขข้อมูล | ระบบกรอกตามข้อมูลต้นทาง ไม่มีการเปลี่ยนแปลงค่า |

## 8. สรุปความคุ้มค่า

| ด้าน | ก่อน AutoNAP | หลัง AutoNAP | ผลลัพธ์ |
|------|-------------|-------------|---------|
| **เวลา** | 5-8 นาที/คน | 15-40 วินาที/คน | ลด **92-97%** |
| **ความถูกต้อง** | 85-95% | 99-100% | เพิ่ม **5-15%** |
| **ความทันเวลา** | ล่าช้าวัน-สัปดาห์ | ภายในวันเดียวกัน | **near real-time** |
| **ความครบถ้วน** | 90-95% | 100% | ไม่มี record หลุด |
| **ภาระงาน** | 4-8 ชม./วัน | 2 นาที/batch | ลด **~99%** |
| **กำลังคนที่ประหยัด** | — | — | **5-8 FTE/ปี** (13 sites) |

> AutoNAP เปลี่ยนงาน data entry ที่ซ้ำซาก ใช้เวลามาก และเสี่ยงผิดพลาด ให้กลายเป็นกระบวนการอัตโนมัติที่เร็ว ถูกต้อง และทันเวลา — เจ้าหน้าที่สามารถใช้เวลาที่ประหยัดได้ไปกับการให้บริการผู้รับบริการโดยตรง
